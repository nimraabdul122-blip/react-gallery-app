# react-gallery-app

A simple and responsive image gallery application built with React and Tailwind CSS. The app fetches dynamic images from the Picsum API and includes clean pagination to browse through different pages.

##  Features

* **Dynamic Images:** Fetches fresh images automatically using the Picsum Photos API.
* **Pagination:** Smooth "Next Page" and "Previous Page" buttons to navigate content.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens using Tailwind CSS.
* **Hover Effects:** Smooth zoom animations when hovering over gallery cards.
* **Lazy Loading:** Images load efficiently to save bandwidth and improve performance.

## Tech Stack

* **Frontend:** React (Hooks: `useState`)
* **Styling:** Tailwind CSS
* **API:** Picsum Photos

##  Project Structure

```text
├── Gallerapp.jsx     # Main gallery component containing the state and UI
└── main.jsx          # Application entry point
```

## How to Run This Project Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate into the project directory:**
   ```bash
   cd react-gallery-app
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
